Projects

DTMS

Dockyard Transport Management System
DTMS is a uber-like service made specially for the vehicles and employees who worked in the Dockyard PLC Colombo. Through this collaborative project, we aim to automate the traditional vehicle requesting method used in the dockyard by providing mobile applications for vehicle service providers and consumers.
For the administration of the dockyard, DTMS provides vehicle tracking, employee status checking, employee management services via a web application.
Technologies
  • Next.js (Based on react)
  • React Native
  • Firebase (Google cloud platform)

PMS

Patient Management System
PMS is a open source software made for storing and managing sensitive data such as patient details of a medical center / hospital securely. Data is being stored encrypted in industry standard AES-256 for maximum security.
This project is started as a request came from Karapitiya Hospital Pediatric ward and Hambantota Hospital Psychiatric ward and currently in alpha stage.
Technologies
  • Next.js (Based on react)
  • Node JS
  • Express JS
  • Postgres
  • GoLang
